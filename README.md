# homebrew-mondo

Homebrew tap for [mondo](https://github.com/zoltanf/mondo) — a power-user CLI for the monday.com GraphQL API.

## Install

```bash
brew install zoltanf/mondo/mondo
```

## Upgrade

```bash
brew upgrade mondo
```

The formula here is regenerated automatically by the release workflow in
[zoltanf/mondo](https://github.com/zoltanf/mondo) on every `v*` tag push.
